# シフト管理システム仕様書 📝

## 1. このシステムについて

このシステムは、アルバイトや仕事のシフト（勤務時間）を簡単に管理するためのウェブアプリケーションです。

- **目的:** 誰が・いつ・何時から何時まで働くのかを、一目で分かるようにする。
- **使う人:** お店の店長さん、アルバイトのリーダー、シフトを管理する人
- **できること:**
    - 新しいシフトの登録
    - 登録されたシフトの一覧表示
    - シフトの変更（編集）
    - シフトの削除

## 2. 主な機能

このシステムには、大きく分けて4つの機能があります。

### ① シフトの登録 🆕

新しいシフトを登録する機能です。

| 入力項目 | 説明 | 例 |
| :--- | :--- | :--- |
| **氏名** | 働く人の名前を入力します。 | `山田 太郎` |
| **日付** | 働く日を選びます。 | `2025-11-21` |
| **勤務開始** | 仕事が始まる時間を入力します。 | `09:00` |
| **勤務終了** | 仕事が終わる時間を入力します。 | `17:00` |

入力が終わったら「＋ 追加」ボタンを押すと、シフトが登録されます。

### ② シフトの一覧表示 👀

登録されているシフトを日付ごとにまとめて表示します。

- **日付:** 「11月21日(金)」のように、分かりやすく表示されます。
- **シフト内容:** その日に働く人全員のシフトが表で表示されます。
    - **氏名**
    - **開始時間** と **終了時間**
    - **勤務時間**（何時間働くか）
    - **操作ボタン**（編集・削除）

### ③ シフトの編集 ✏️

登録済みのシフト情報を変更する機能です。

1.  変更したいシフトの横にある「編集」ボタンを押します。
2.  一番上の入力フォームに、そのシフトの情報が自動で入力されます。
3.  変更したい部分（時間など）を修正します。
4.  「✓ 更新」ボタンを押すと、情報が更新されます。

### ④ シフトの削除 🗑️

シフトを削除する機能です。

1.  削除したいシフトの横にある「削除」ボタンを押します。
2.  「このシフトを本当に削除しますか？」という確認メッセージが表示されます。
3.  「OK」を押すと、シフトが完全に削除されます。

## 4. 技術スタック

このシステムは、モダンで一般的に利用されている以下の技術スタックで構築されています。

### フロントエンド

| 技術 | 役割 |
| :--- | :--- |
| **React** | UI（画面の見た目や操作感）を構築するためのJavaScriptライブラリです。コンポーネントベースで効率的な開発が可能です。 |
| **Tailwind CSS** | デザインを整えるためのCSSフレームワークです。クラスを指定するだけで、迅速にモダンなデザインを適用できます。 |
| **HTML5** | Webページの基本的な構造を定義します。 |

### バックエンド & データベース

| 技術 | 役割 |
| :--- | :--- |
| **Supabase** | データベース、リアルタイム機能、認証などを提供するBaaS（Backend as a Service）プラットフォームです。 |
| **PostgreSQL** | Supabaseが内部で使用している、信頼性の高いオープンソースのリレーショナルデータベースです。シフトデータはここに保存されます。 |

## 5. 今後の改善案（もっと良くするために）

このシステムをさらに便利にするために、以下のような機能を追加することが考えられます。

- **ログイン機能:** ユーザー名とパスワードを入力しないと使えないようにして、セキュリティを高める。
- **担当者ごとの色分け:** シフト表で、人ごとに色を変えてもっと見やすくする。
- **通知機能:** 新しいシフトが登録されたり、変更されたりしたときに、関係者にメールやチャットでお知らせする。
